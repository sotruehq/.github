# SotrueHQ Global GitHub Configuration

This repository contains **shared configuration, templates and workflows** for all repositories in the `sotruehq` organization.

It includes:

- Organization profile (shown on https://github.com/sotruehq)
- Default issue templates (bug reports, feature requests, chores)
- Default pull request template
- Contributing guidelines
- Code of conduct
- Security policy
- Shared GitHub Actions workflows (labelling, stale issues, etc.)

These defaults apply to all public repositories in the organization that do not override them with their own `.github` configuration.
