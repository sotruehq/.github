# SotrueHQ

**Sotrue** is a social OTT platform where creators and studios share long-form content, and viewers discover it through smart, snackable clips.

We are building:

- A scalable Node.js + PostgreSQL backend
- Video processing pipelines for clips and streaming
- Web, mobile and TV apps
- Tools for creators and internal operations

---

## Repositories

Some key repos (names may evolve):

- `sotrue-backend` – Core APIs and services
- `sotrue-video` – Video processing pipeline
- `sotrue-web` – Web OTT client
- `sotrue-admin` – Admin and operations panel
- `sotrue-mobile` – React Native mobile app
- `sotrue-tv` – TV client
- `sotrue-shared` – Shared types, SDK and UI packages

Most code is currently private and under active development.
